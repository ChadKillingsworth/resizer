Plugins:
PrettyGIFs - Octree quantization and dithering for GIFs and 8-bit PNGs
PsdReader - Convert PSD files to jpeg, png, or GIF on the fly
PsdComposer - Re-compose PSD files on-the-fly. Rewrite text layers, hide/show layers. Minimal support for text layer effects.
AnimatedGifs - Resize GIF movies on the fly. PrettyGIFs plugin suggested. Slow on first request.
DiskCache - Caches resized images to disk for huge speed gains, using invisible request rewriting.
S3Cache - Caches resized images to S3 for scalability - but uses redirects.
S3Backend - Loads files from an amazon S3 backend.
ZipBackend - Can read images from Zip files.
